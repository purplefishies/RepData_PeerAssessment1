Are there differences in activity patterns between weekdays and weekends?

We will create a new dataset from the altered dataset and create a new column for the type of day being either a weekend or a weekday.

newaltered <- altered %>% 
  mutate(type_of_day=ifelse(weekdays(as.POSIXlt(date)) %in% c("Sunday","Saturday"),"weekend","weekday"))

Next we will create a new variable that contains the group by interval for each type of type of day to determine how the activity is affected by whether we are in a weekend or not.

per_type_of_day <- newaltered %>%
